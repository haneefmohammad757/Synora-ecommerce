# 🛒 Synora - Full Stack E-Commerce Platform

Synora is a modern full-stack e-commerce web application built using **React**, **Spring Boot**, and **MySQL**. It provides a complete online shopping experience with secure authentication, product management, shopping cart, wishlist, online payments, and an admin dashboard.

---

## 🌐 Live Demo

**Frontend:**  
https://synora-ecommerce.vercel.app

**Backend API:**  
https://synora-backend-1.onrender.com

---

## 🚀 Features

- User Registration & Login
- JWT Authentication
- Admin Dashboard
- Product Management (Add, Update, Delete)
- Hero Product Slider
- Product Search
- Product Sorting
- Category Filtering
- Shopping Cart
- Wishlist
- Order Management
- Razorpay Payment Integration
- Responsive Design
- Mobile Friendly UI

## ⚙️ Getting Started

### Clone the Repository

```bash
git clone https://github.com/haneefmohammad757/Synora-ecommerce.git
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

### Backend

```bash
cd backend
./mvnw spring-boot:run
```
